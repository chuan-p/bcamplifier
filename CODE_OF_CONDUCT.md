# Code Of Conduct

## Our Commitment

We want Bandcamplifer to be a welcoming project for contributors with different backgrounds, experience levels, and perspectives.

## Expected Behavior

Please:

- be respectful and constructive
- assume good intent and ask clarifying questions before escalating
- keep technical disagreement focused on the work, not the person
- share feedback in a way that helps others improve

## Unacceptable Behavior

This project does not accept:

- harassment, discrimination, or intimidation
- personal attacks, insults, or bad-faith provocation
- publishing private information without permission
- disruptive behavior that repeatedly derails collaboration

## Reporting

If you experience or witness behavior that violates this code, contact the maintainer through GitHub so the issue can be reviewed and handled privately when appropriate.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors from project spaces when needed to protect the community and keep collaboration productive.
