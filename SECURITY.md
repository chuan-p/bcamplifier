# Security Policy

## Supported Versions

Security fixes are applied to the current main branch.

## Reporting A Vulnerability

Please do not post security-sensitive reports in a public issue.

Instead:

1. Use GitHub's private vulnerability reporting for this repository if it is enabled.
2. If private reporting is not available, contact the maintainer through GitHub before sharing details publicly.

When possible, include:

- the affected Bandcamplifer version
- steps to reproduce the issue
- impact on Bandcamp sessions, account actions, or stored data
- any suggested mitigation or patch direction

## Disclosure

We will try to confirm the report, assess the impact, and coordinate a fix before public disclosure.
