# Privacy Policy

BC Amplifier is a browser extension and userscript that enhances the Bandcamp feed.

## Single purpose

The extension has one purpose: improve the Bandcamp fan feed with richer release context, playback controls, and Bandcamp-native shortcut actions. It is not designed to monitor browsing outside Bandcamp, and its content scripts are limited to Bandcamp feed, album, and track pages.

## What the extension does

The extension loads Bandcamp pages in the browser to extract release metadata, render richer feed cards, support inline playback controls, and provide shortcut actions such as wishlist and buy.

## Data the extension processes

The extension may access or process:

- Bandcamp page URLs needed to load release information
- page content from Bandcamp release pages needed to extract metadata such as tracklists, release dates, artist names, and locations
- local extension settings and cached metadata stored in browser storage

## What the extension does not do

- It does not send data to a separate developer-owned backend
- It does not sell personal data
- It does not use third-party analytics, advertising SDKs, or tracking beacons

## Network requests

The extension makes network requests to Bandcamp domains in order to load release information and support the extension's features. These requests are limited to the functionality of the extension.

Supported domains:

- `https://bandcamp.com/*`
- `https://*.bandcamp.com/*`

The extension does not send processed data to a separate developer-owned backend. Network activity is limited to Bandcamp requests needed for extension features.

## Storage

The extension stores settings and cached release metadata locally in the browser to improve performance and preserve user preferences.

## User-initiated actions

Some features can affect the user's Bandcamp account only after the user explicitly clicks them, such as:

- wishlist actions
- opening Bandcamp's native purchase flow

The extension does not automatically complete purchases.

## Incognito mode

The extension is intended for normal signed-in browsing sessions on Bandcamp. It should remain disabled in incognito mode unless a user explicitly decides they need it there.

## Data sharing

The extension does not share user data with the developer or unrelated third parties. Data is only transmitted to Bandcamp as needed to perform the extension's features inside the user's browsing session.

## Changes to this policy

This policy may be updated as the extension changes. Any public store listing or hosted policy page should be updated to match the current extension behavior.
